# Complaint Registration System

## 🚀 Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL
- Cloud: Cloudinary

## ✨ Features
- User complaint submission
- Admin dashboard
- Department handling
- Image upload

## ⚙️ Setup Instructions

### Backend
cd Complaint_register
mvn spring-boot:run

### Frontend
cd complaint_registration
npm install
npm run dev
